Le but de ce devoir est de créer un site web qui permet de gérer des utilisateurs.

Le site doit permettre de :

  • Créer un utilisateur
  • Modifier un utilisateur
  • Supprimer un utilisateur
  • Afficher la liste des utilisateurs

Les utilisateurs doivent avoir les champs suivants :

  • Nom
  • Prénom
  • Age
  • Adresse
  • Code postal
  • Ville
  • Numéro de téléphone
  • Adresse email

Le site doit être sécurisé contre les injections SQL.

Le site doit être responsive.

Le site doit être codé en Python.

Le site doit utiliser une base de données.

Le site doit utiliser le framework Django.

Le site doit être hébergé sur un serveur.
